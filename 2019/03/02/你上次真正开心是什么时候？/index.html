你上次真正开心是什么时候？

发表于 2019-03-02 | 分类于 随笔 | | 本文总阅读数 次
字数统计: 1.1k | 阅读时长 ≈ 3

  开学前和学长出去喝酒。

  酒过三巡，开始聊到最近的状况。

  “没什么不开心的，也没什么开心的。”

  学长说着，又倒了一杯酒。

  他前阵子刚找到一份大厂的工作，遇到的上司不错，和同事之间的关系也算融洽，但不知道为什么——

  “就是提不起劲来，感觉没意思。”

  不止是学长。

  那天一起喝酒的老刑和蛋蛋也有这个感受。

  老刑刚找到一份投行的实习，蛋蛋在和朋友一起接外包。

  平时在朋友圈里，他们发的生活片段或文艺或高大上，看上去活得充实而有趣。

  但他们大部分时间其实也不怎么开心。

  “有时候我也怀疑是不是自己太作了。明明现在做的是自己喜欢做的事情，之前也一直在为能以此谋生而努力。但真的做了一段时间却觉得：

  啊，日子过得好慢呀。”

  老邢叹道。

  “所以，你上次真的感到开心是什么时候？”

  在被学长反复问这个问题的时候，我愣了大概两分钟。

  那两分钟里，我试图搜寻出关于快乐的记忆，但时间线却好像都有点久远。

  在占比很大的记忆里，充斥着无聊、焦虑、孤独，以及琐碎的因为看剧、刷图片而引起的笑意。

  事实上，在学长问我之前，我也已经隐约有了这样的感觉：

  好像，现在越来越难开心了。

  每本好看的小说快结束时，会极度惋惜和害怕，为了避免陷入长久的空虚里，要赶快去找下一本。

  在朋友圈里刷到有趣的段子后，会不由自主地继续刷新下去来消磨时间。

  完成一个项目后，希望能马上开始下一个项目，以求获得价值和存在感。

  一切仿佛陷入了一个无解的死循环。

  不过，很奇怪的，当我意识到自己其实“没那么开心”时，反倒觉得松了口气。

  就像是之前一直觉得身体不对劲，但不知道“出毛病”的地方在哪里，去医院做了检查才知道，原来是哪里和哪里出了问题。

  事实上，最可怕的可能是，很多时候，我们都没意识到——

  原来自己没觉得太开心。

  如果一切都很正常，一切也都往好的方向发展，那么，好像没有理由不开心才对？

  但不是的。

  一切都正常只能让我们可以避免“痛苦”。如果没那么开心的话，可能意味着，现在的生活缺少了目标和新鲜感。

  而只有意识到这些，我们才能给自己去创造这些东西，从而让自己快乐。

  我们都太轻视“开心”这件事了。

  或者说，我们都太轻视“好好生活”这件事。

  我们很少去问自己：

  你今天开心吗？

  你今天过的好吗？

  于是我们只是浑浑噩噩地去起床，浑浑噩噩地上床。

  然后一晃眼，几十年就这样过去了。

  那么不妨，从明天开始认真审视自己，并保持新鲜和很好奇：

  送自己一朵好看的花，学做一个好吃的甜点，看原来没看过的书，走原来没走过的路，去以前没去过的小店逛逛，认识完全没接触过的人。

  就像是《今天也要用心过生活》里，松浦弥太郎说的：

  只要日子过得新鲜有趣，人就有仔细过生活的动力。

  在做完这些后，即使没有觉得快乐，也一定变成了一个更懂得生活的人了。

  然后，一定有一天，快乐会自己找上门来。

  “你上次真正开心是什么时候？”

  “上一秒。”
